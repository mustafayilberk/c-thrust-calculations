# C Thrust Calculations

Simple C programs to calculate motor thrust and classify VTOL motors as 
weak, medium, or strong based on RPM and motor constant (k).

## Files
- `thrust_hesaplama.c` – Basic thrust calculation
- `thrust_hesaplama_siniflandirma.c` – Thrust + motor class (ZAYIF / ORTA / GUCLU)

This is part of my journey to become an avionics & embedded systems engineer.
